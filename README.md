***# Fitness Management System***



***A simple Fitness Management System developed in C++.***



***Project Objective:***

***This project helps users manage fitness information, BMI, workout plans, and diet recommendations.***



***Features:***

***- Add Member***

***- View Member***

***- BMI Calculator***

***- Workout Recommendation***

***- Diet Recommendation***



***Installation:***

***1. Clone repository***

***2. Open project***

***3. Compile main.cpp***

***4. Run executable***
